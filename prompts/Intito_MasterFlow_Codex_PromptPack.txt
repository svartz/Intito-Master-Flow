
Codex Prompt Pack – Intito MasterFlow

1. Initialize project
Read docs/architecture and object catalog. Create TM1 objects:
IMF.D.Dimension, IMF.D.Version, IMF.C.Config, IMF.C.Version.

2. Implement Version.Create
Create process IMF.P.Version.Create supporting clone from master.

3. Implement CloneFromMaster
Copy elements, hierarchies and attributes to work dimension.

4. Implement Diff Engine
Build IMF.P.Compare.VersionToMaster and store results in IMF.C.Diff.

5. Implement Validation
Create IMF.P.Validate.Technical with attribute and hierarchy checks.

6. Implement Impact Analysis
Create IMF.P.Impact.RunAll checking cube and subset dependencies.

7. Implement Publish
Create IMF.P.Commit.PublishVersion including archive logic.

8. Implement Archive
Create IMF.P.Commit.ArchiveMaster creating timestamped archive dimensions.

9. Implement Rollback
Create IMF.P.Rollback.ToArchive restoring archived dimension.

10. Implement Export Engine
Create export processes for CSV ParentChild and LeafFlat formats.
