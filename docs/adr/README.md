# IMF ADR

This folder is reserved for Architecture Decision Records related to IMF.

Use ADRs when:

- a design choice needs long-term traceability
- an implementation path was chosen over another
- release/build/deployment policy changes
- Bedrock adoption decisions change

Suggested naming:

- `ADR-0001-bedrock-adoption.md`
- `ADR-0002-release-build-policy.md`
